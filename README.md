# AI Live Transcription System

This is a real-time speech-to-text desktop application built using Python and Whisper.

## Features

* Real-time transcription
* Offline AI processing (Whisper)
* Multi-language support
* Dual-panel UI (Live + Final output)

## Tech Stack

* Python
* Tkinter
* Faster-Whisper
* SoundDevice

## How to Run

pip install faster-whisper sounddevice numpy
python live_transcription_system.py
